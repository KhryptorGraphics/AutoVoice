# Specification: Live Karaoke Voice Conversion

**Track ID:** live-karaoke_20260124
**Type:** Feature
**Created:** 2026-01-24
**Status:** Draft

## Summary

Real-time karaoke voice conversion system where users upload a song, the system separates vocals from instrumental, and as the user sings along karaoke-style, their voice is converted in real-time to sound like the original artist with all vocal techniques (melisma, coloratura, vocal runs, bravato) preserved.

## Context

AutoVoice is a GPU-accelerated singing voice conversion system running on NVIDIA Jetson Thor. The recently completed SOTA pipeline refactor provides the core components: MelBandRoFormer vocal separation, StreamingConversionPipeline for real-time processing, ContentVec content extraction, RMVPE pitch extraction, CoMoSVC decoder, and BigVGAN vocoder. This feature integrates these components into a complete live performance experience with a web interface.

## User Story

As a music enthusiast, I want to sing karaoke and sound like my favorite artist so that I can experience performing with professional vocal ability.

### Audio Routing Design

- **Speakers (audience output)**: Instrumental track + user's converted voice (sounds like original artist)
- **Headphones (user output)**: Original song with artist vocals (so user can follow along and sing in sync)

This allows the performer to hear and follow the original while the audience hears them performing with the artist's vocal capabilities.

## Acceptance Criteria

- [ ] User can upload any song and vocals are separated within 30 seconds
- [ ] Real-time conversion latency is under 50ms end-to-end
- [ ] Audio routing supports separate headphone/speaker outputs
- [ ] Audio outputs are configurable (select which device gets which stream)
- [ ] Web interface hosted at autovoice.giggadev.com with HTTPS
- [ ] System preserves vocal techniques (melisma, coloratura, vocal runs)
